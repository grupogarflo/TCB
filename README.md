# cancunbay



## Getting started

Cancun bay 2023

backEnd folder- php laravel 7
cmsSite  - Nuxtjs over backEnd/public/cms folder
frontend - Nuxtjs over backEnd/public folder

to deploy project:

```
clone project on server
install composer dependencies 
create env file on backEnd
run php artisan migrate 
run php artisan seed
on your local environment run "npm run generate" to create dist folder on cmsSite 
repeat the previous step on frontend folder 
upload cmsSite dist folder and frontend dist folder on Server by FTP

```
